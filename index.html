Accessible Web Development

Welcome to our web development workshop and study designed for students that are blind!

This course will be held between 2:30pm and 5:00pm, on Saturdays between August 22nd and October 10th. We will move through these materials over 8 sessions in pace with student learning.

Office Hours and Additional Supports

Addditional help will be available throughout the workshop on an appoinment basis. Please email claire.kv(at)nyu(dot)edu to book an office hour appointment. If you must miss a session, please let us know and we can arrange a check-in. Additional remote work sessions will be held on Sundays toward the end of the course (Exact dates TBD).

CSS and design units will be supported by the use of downloadable .pdfs and .ai files of tactile manipulatives. Students were asked to complete a survey pre and post, to help us improve the course.

Lesson Outline

  1. Introduction
    • Intro to the web: the net, browsers, web languages, websites, file structure
    • User Experience: usability and accessibility
    • Discussion of text editors and other software to be used.
  2. HTML
    • HTML syntax, basic elements, and our first site
    • New elements, attributes, and mulitple pages
    • Semantic HTML Elements
  3. CSS and Design
    • Intro to visual styling, adding CSS, syntax, selectors, web colors, font types/sizes, and the box model
    • CSS review
  4. JavaScript
    • Intro to JavaScript: Inputs/interactions, syntax, functions, variables, and ARIA
    • Uploading a site to a server: File transfer prototcol (FTP)
