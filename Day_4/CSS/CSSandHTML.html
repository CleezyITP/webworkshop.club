Incorporate CSS into Your HTML

Once you've added a link to an extrernal stylesheet, there are a couple ways to incoprorate your styles into your HTML.

  • Element selectors : any HTML element you have created a rule for will be applied to the HTML elements autoatically
  • class selectors : class selectors you have defined in your CSS file can be referred to in your HTML with the class attribute. If you have a class selector defined as 
    .myBigClass { 
    font-size:72px;
}
    you would call it in your code as 
    <p class="myBigClass">
  • I D selectors : an I D selector is called once. I D selectors you have defined in your CSS file can be referred to in your HTML with the id attribute. If you have a I D selector defined as 
    #banana { 
    background-color:yellow;
}
    you would call it in your code as 
    <div id="banana">
