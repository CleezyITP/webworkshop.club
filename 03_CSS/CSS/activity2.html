CSS/Design Activity 2

Let's try and pull together some new ideas into your CSS file. First, develop a color palette/scheme for your site based on color theory descriptions. This palette is 100% up to you to create, but please be prepared to describe your rationale, and test your foreground and background colors using either The Level Access Contrast Checker (accepts RGB and Hex values) or Webaim's contrast analyzer (Hex only). Use Divs, and your existing HTML elements to implement your new color scheme.
