Debugging refers to different methods and tools for finding errors in your code...

  1. Check for spelling and syntax errors - Line by line, character by character (errors are often a missing characters)

  3. Enter the following code into the beginning of your script: 
    window.onerror = showerror;
function showerror(msg, url, line, column, rerr){
 alert('Error: ' + msg + ' Line: ' + line + ' character: ' + column + ' url ' + url + ' ' + rerr);
 window.onerror=null;  
 return true;  
}
