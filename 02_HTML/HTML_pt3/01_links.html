Review of Attributes and Links

Attributes

As a refresher, attributes are used to modify elements. Attributes are used in a number of ways and are useful when adding links (href), images (src), accessibility (alt), adding style with CSS and interactions with Javascript (id, class - we will learn more about later). previously, we used the reference attribute (href) to create links with the anchor element.

Links

Links are created by using the anchor element. There are many different kinds of links you can make. For example, you can link to an email address, an image, or an email service.
